Coursework

Throught my academic career, I completed several key projects. I learned new skills from every one of these projects, and I still use some of them to this day.

Analyticks


Summary: A group software development project completed over the course of 10 weeks. Sponsored by Deutsche Bank, the aim was to produce a web application to help them monitor stock changes. My roles in the team were to organise group meetings, create and compile the group reports, and to help out with visual components used for marketing.

Outcome: Our team ultimately came joint first out of the year group. We were praised largely on our visual materials, for creating a professional looking product.

Skills learned:
  • Report writing
  • Organising multiple team members
  • Working to deadlines
  • Responding to feedback
  • Presentation
Tools used:
  • LaTeX
  • MS Office
  • HTML/CSS (Bootstrap)
  • Dropbox
  • BitBucket

This Little Piggy


Summary: A paired project to create a game using 3D and 2D graphics. The game is to raise a virtual piglet from birth to bacon. My role was to work on the interface for the game, as well the backend gameplay logic.

Skills learned:

  • Gameplay design

Tools used:

  • Javascript
